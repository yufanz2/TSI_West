# TSI_West
This repository includes maps I developed for TSI study west area.
"Raster" and "Shapefile" are ArcGIS Pro Layer Package.
"Raster" includes maps showing flood susceptibility & vulnerability index.
"Shapefile" includes maps with attribute table data for a better visulization.
"Data sources" shows the sources of all the original data used to develop flood susceptibility and vulnerability index.
"Codes" are codes used in "Adding Attribute" tool in ArcGIS Pro to automate the identification of hydrologic soil groups, Curve Numbers, and other relevant attributes.
